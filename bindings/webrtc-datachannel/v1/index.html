ShadowClaw A2A WebRTC DataChannel Binding

Binding URI
https://xt-ml.github.io/shadow-claw/bindings/webrtc-datachannel/v1
Protocol version
1.0
Transport address scheme
webrtc://<peer-id>

AgentInterface

interface AgentInterface {
  url: string; // e.g. "webrtc://<peer-id>"
  protocolBinding: "https://xt-ml.github.io/shadow-claw/bindings/webrtc-datachannel/v1";
  protocolVersion: "1.0";
}

ShadowClaw Usage

  • url is the logical peer address for this binding. In ShadowClaw, that is a WebRTC-style address such as webrtc://peer-123.
  • protocolBinding declares the transport binding type: A2A over WebRTC DataChannel.
  • protocolVersion is the A2A version supported by this interface.
